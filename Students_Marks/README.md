Students Marks Calculator – C# Guided Project

This project is a console-based C# application created as part of the Microsoft Learn Guided Projects. It demonstrates how to use basic programming concepts in C# to collect student information, calculate marks, and display results.

The application allows users to enter student names and their marks, then processes the data to compute totals and averages.

📌 Project Overview

The Students Marks Calculator is a simple program that simulates how a system can manage and process student marks.

The program performs the following operations:

Accepts student names

Accepts marks for each student

Calculates total marks

Calculates average marks

Displays formatted results in the console

This project focuses on practicing basic C# programming and logic building.

⚙️ How the Program Works

The program stores a list of student names.

It asks the user to input marks for each student.

The program calculates the total marks.

It calculates the average marks of all students.

The results are printed to the console in a clear format.
