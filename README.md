C# Guided Projects – Microsoft Learn

This repository contains my solutions and practice projects completed while following the C# Guided Projects from Microsoft Learn. The goal of this repository is to document my progress, strengthen my understanding of C# programming, and build hands-on experience with real coding scenarios.

The projects included here are based on structured exercises provided by Microsoft Learn, where learners build small applications step-by-step to understand programming concepts through practical implementation.

📚 About This Repository

This repository serves as a learning portfolio for C# development.
It contains guided coding exercises, mini-projects, and practice implementations completed while learning C# fundamentals.

Through these projects, I practice:

Core C# syntax

Object-Oriented Programming (OOP)

Problem-solving using C#

Writing clean and maintainable code

Building small real-world style console applications

Each project folder contains the implementation of a specific guided exercise along with the source code.

🚀 How to Run the Projects

Clone the repository

git clone https://github.com/Tashyak/Guided_Projects.git

Navigate to the project folder

cd Guided_Projects

Run the program using the .NET CLI

dotnet run

You can also open the project in Visual Studio or VS Code and run it directly.
